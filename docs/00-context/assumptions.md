# Assumptions

1. User has Alpaca paper trading account (free)
2. User has NewsAPI free tier key (500 requests/day)
3. User has Reddit API credentials (PRAW)
4. Bot runs on local machine (Mac) — not cloud-deployed initially
5. Trading US equities only (no options, no crypto initially)
6. Market hours: 9:30 AM - 4:00 PM ET (extended hours optional later)
7. Paper account size: $50,000
8. Internet connection is stable
9. SQLite is sufficient for logging (no need for Postgres initially)
